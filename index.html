prediction

This project develops a machine learning model to estimate the selling price of vehicles based on factors like make, model, year, mileage, and condition. January, 2024

SMS spam
Detection

In this project, I developed a machine learning model to differentiate between spam and legitimate messages. By collecting a dataset comprising examples of both spam and non-spam (ham) messages, I trained the model using advanced techniques like natural language processing (NLP) and classification algorithms. The model learned to recognize patterns and features in the text, enabling it to accurately classify incoming messages as either spam or ham. This project aimed to enhance user experience and security by automatically filtering out unwanted spam messages in real-time.
